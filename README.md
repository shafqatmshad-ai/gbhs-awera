# 🎓 GBHS Awera – Student Registration & Result System

This project is a simple and responsive **School Web Application** developed for **Government Boys High School Awera Bagh (AK)**.
It provides features for **student registration** and **result (gazette) display/download**.

---

## 🌐 Live Website

👉 https://shafqatmshad-ai.github.io/gbhs-awera/

---

## 🚀 Features

* 📝 Student Registration Form
* 📄 Gazette (Result PDF) Download
* 🏫 School Header with Logo & Details
* 📱 Responsive Design (Mobile Friendly)
* 🖨️ Print-ready Form Layout
* 📂 Organized file structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages (Hosting)

---

## 📁 Project Structure

```bash
gbhs-awera/
│
├── index.html              # Main homepage / registration form
├── style.css               # Styling (layout + design)
├── script.js               # Form logic / validation (if used)
│
├── images/                 # Logo & assets
│   └── logo.png
│
├── docs/                   # Gazette / result PDFs
│   └── result-2026.pdf
│
└── README.md               # Project documentation
```

---

## 📥 How to Use

1. Open `index.html` in browser
2. Fill the registration form
3. Download gazette from button
4. Deploy using GitHub Pages
